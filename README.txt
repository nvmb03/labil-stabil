LABIL–STABIL: Tragische Superhelden-Sterne
==========================================

KONZEPT:
Superhelden als abstrakte Sterne. Im entscheidenden Moment 
versagen ihre Superkräfte - labile Momente des Scheiterns.

ANLEITUNG:
1. Öffnen Sie index.html für das Menü (empfohlen)
2. Oder öffnen Sie einzelne Dateien direkt:
   - spiderman.html (ADHS/Hyperfokus)
   - superman.html (Sensorische Überlastung)
   - hulk.html (Impulskontrolle)
   - batman.html (Perfektionismus)
   - flash.html (Entscheidungslähmung)

STEUERUNG:
- Maus bewegen: Stern folgt der Maus
- Mausklick: Spezialfähigkeit aktivieren (Hulk, Batman)
- Beobachten Sie die zufällig auftretenden labilen Momente!

STATUS-ANZEIGE:
- Grün = Stabil (Superkraft funktioniert)
- Rot = Labil (Superkraft versagt)

TECHNIK:
- p5.js (wird automatisch aus dem Internet geladen)
- Keine Installation nötig
- Funktioniert in Chrome, Firefox, Safari, Edge
- Benötigt Internetverbindung (für p5.js Library)

OFFLINE-VERSION:
Falls kein Internet verfügbar ist, liegt p5.min.js 
im Ordner und wird automatisch verwendet.

Entwickelt mit: JavaScript, p5.js, HTML5, CSS3
