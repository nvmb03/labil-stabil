# Labil–Stabil: Tragische Superhelden-Sterne

Interaktive Installation für Gestaltung mit Code.

## Konzept

Superhelden als abstrakte Sterne, die durch Bewegung ihre Identität offenbaren – doch im entscheidenden Moment versagen ihre Kräfte. Das Thema "Labil–Stabil" manifestiert sich im Kontrast zwischen stabiler Superhelden-Identität und labilem Moment des Scheiterns.

## Superhelden

- **Spider-Man**: ADHS/Hyperfokus - Reizüberflutung, Web-Spam
- **Superman**: Autismus/Sensorische Überlastung - Shutdowns, Meltdowns
- **Hulk**: Impulskontrollstörung - Überreaktion, Frustrations-Spirale
- **Batman**: Zwangsstörung - Perfektionismus-Loops, ungewolltes Auftauchen
- **Flash**: Entscheidungslähmung - Paralyse trotz Geschwindigkeit

## Technik

- p5.js (JavaScript)
- Keine Installation nötig
- Läuft direkt im Browser

## Verwendung

1. `index.html` öffnen für Menü
2. Oder einzelne Dateien direkt öffnen

## Steuerung

- Maus bewegen: Stern folgt
- Mausklick: Spezialfähigkeit (bei Hulk, Batman)
- Beobachten: Labile Momente
